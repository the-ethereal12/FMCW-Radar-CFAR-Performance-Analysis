# 📡Performance Evaluation of FMCW Radar with CFAR Detection under Low-SNR and Clutter Conditions
## 📌 Project Overview

This project presents a complete simulation of an FMCW (Frequency Modulated Continuous Wave) Radar System using Python.  
It implements the full radar signal processing chain including signal generation, target modeling, range detection, velocity estimation, adaptive thresholding, and target tracking.

The system is capable of detecting and tracking multiple moving targets under noisy conditions using advanced signal processing techniques.

---

## 🎯 Objectives

- Simulate FMCW chirp signal generation  
- Perform Range Estimation using FFT  
- Generate 2D Range-Doppler Maps  
- Implement CFAR-based adaptive detection  
- Evaluate detection performance using ROC curves  
- Track moving targets using Kalman Filter  
- Analyze Radar Cross Section (RCS) effects  
- Simulate multi-target environments  

---

## 🧠 System Architecture

The project follows these major signal processing stages:

1. FMCW Chirp Generation  
2. Target Reflection Modeling  
3. Beat Signal Formation  
4. Range FFT Processing  
5. Doppler FFT (2D FFT)  
6. CFAR Detection  
7. ROC Performance Analysis  
8. Kalman Filter Tracking
